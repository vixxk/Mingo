# TALK-TO-ME

A project with a Node.js Express backend and a React Native Expo frontend.

## Project Structure

- `backend/`: Node.js Express server with MongoDB and Redis.
- `frontend/`: React Native application using Expo.

## Getting Started

### Backend
1. `cd backend`
2. `npm install` (If not already done)
3. `cp .env.example .env` (Configure your environment variables)
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install` (If not already done)
3. `npm start`
